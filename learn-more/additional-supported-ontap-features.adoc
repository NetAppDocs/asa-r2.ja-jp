= ASA r2 システムでサポートされる一般的なONTAP機能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ASA r2 システムはONTAPの合理化されたバージョンを実行するため、多くの一般的なONTAPタスクと System Manager 機能は、 ASA r2 システムでも他のONTAPシステムと同じように実行されます。

共通の機能の詳細については、次のONTAPドキュメントを参照してください。
